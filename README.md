# Red Bus Travellers Database (SQL Project)

- Designed a database for Red Bus Travellers using SQL.

- Created PASSENGER and PRICE tables to store travel and fare details.

- Inserted sample data representing real booking records.

- Wrote queries to filter passengers, calculate fares, and retrieve specific information.

- Used JOIN operations to match bus type and distance for price calculation.

- Performed UPDATE, DELETE, and DROP operations to manage data and structure.

- Demonstrated core SQL concepts such as table creation, data manipulation, and transactions.
